# Handoff: Creo Study 三田教室 トップページ リニューアル（3案）

## Overview
学習塾「クレオスタディ」三田教室のトップページを刷新するデザイン提案です。既存サイト（creo-study.com）のロゴ・世界観は維持しつつ、見た目を中心とした構成に書き換え、以下を前面に押し出しています。

- 指導方針：少人数グループ指導 + 個別指導で、一人ひとりに合った最適な学びを提供
- 特色講座：アビリティプログラム／パズル道場／読み解く国語
- クレオスタディの想い・特徴

上部ナビゲーションはリンクのみのシンプル構成（採用情報・ブログ・受験体験記は今回スコープ外）。姉妹校である上野入谷教室は、フッターからの外部リンク1本のみで案内しています（`https://www.creo-study.com/ueno` — 既存の実ページ）。

**3案は同じ1ページ（同じ構成・同じコピー）に対する、3つの異なる視覚的方向性です。** クライアントが方向性を選定するための比較材料であり、3つとも本番実装する想定ではありません。どれか1案が選ばれたら、その1案だけを実装してください。

## About the Design Files
このバンドルに含まれるHTMLファイルは **デザインリファレンス（参考実装）** です。本番のコードとしてそのまま流用するものではありません。既存のプロダクトコードベース（React / Vue / SwiftUI / ネイティブなど）がある場合は、そのコードベースが既に採用しているパターン・コンポーネント・ライブラリを使って、このデザインを再構築してください。まだ環境がない場合は、プロジェクトに最も適したフレームワークを選定した上で実装してください。

HTMLファイルはすべて **インラインスタイル** で書かれているため、各要素の`style`属性から色・余白・フォントの値をそのまま読み取れます（クラス名やCSS変数には依存していません）。

## Fidelity
**High-fidelity** — 色・タイポグラフィ・余白・レイアウトは以下に記載の値で確定しています。ピクセル単位で正確に再現してください。

## Page Structure（3案共通の構成）
1. **Header** — ロゴ + ナビリンク4つ（指導方針／特色講座／クレオスタディについて／お問い合わせ）。すべて同一ページ内アンカーリンク。
2. **Hero** — バッジ（学習塾名）、H1見出し（2行）、本文（3行）、CTA2つ（主：お問い合わせ／副：無料体験・資料請求）、ヒーロービジュアル（写真プレースホルダー）。
3. **指導方針セクション**（id: `pt-policy`）— アイキャッチラベル、H2、リード文、2パネル（少人数グループ指導／個別指導：それぞれアイコン＋見出し＋本文）、下部に対象学年・受験対応の注記1行。
4. **特色講座セクション**（id: `pt-programs`）— アイキャッチラベル、H2、リード文、3カードグリッド（アビリティプログラム／パズル道場／読み解く国語：それぞれ番号＋タグ＋見出し＋本文）。
5. **クレオスタディについてセクション**（id: `pt-about`）— アイキャッチラベル、H2（2行）、本文3段落、太字の結びの一文。3案とも背景を反転（濃色地に明るい文字）した対比セクション。
6. **Footer**（id: `pt-contact`）— CTA見出し＋本文＋ボタン2つ、三田教室の詳細（住所・電話・受付時間・最寄り駅）、姉妹校（上野入谷教室）への外部リンク、下部バー（小ロゴ＋コピーライト）。

## Pattern A — やわらかな余白（Soft Light / Airy Warmth）
**Concept**: 丸みと余白で、やさしく迎える。ロゴの4色（金・珊瑚・水色・緑）をそのまま柔らかく使う、あたたかい印象。

**Colors**
- 背景: `#FBF8F3`（生成り）／ カード面: `#FFFFFF`
- 本文テキスト: `#3A3530`（メイン）／ `#655D53`, `#6B6154`（サブ）／ `#948C81`, `#7A736A`（注記・キャプション）
- 罫線: `#EEE6D8`
- ブランドドット（装飾用・原色のまま）: 金 `#F8BC01` ／ 珊瑚 `#F59084` ／ 水色 `#47BEE7` ／ 緑 `#7CC628`
- テキスト用（コントラスト確保のため濃色化）: 金 `#B8860B` ／ 珊瑚 `#D9604E` ／ 水色 `#1A93BE` ／ 緑 `#5C9418`
- パネル淡色背景: 金系 `#FEF3D6`（少人数グループ指導パネル）／ 水色系 `#EAF7FC`（個別指導パネル）
- 反転セクション（クレオスタディについて）: 背景 `#3A3530`、文字 `#FBF8F3`、アクセント `#F8BC01`

**Typography**
- 見出し（H1/H2）: **M PLUS Rounded 1c**, weight 800 のみ使用
  - Hero H1: 50px / line-height 1.5
  - セクションH2: 34px / 1.6（About: 30px/1.7、Footer: 28px）
- 本文: **Noto Sans JP**, weight 400（本文）／ 500（ナビリンク・アイキャッチラベル）／ 600（小見出しラベル・CTAボタン）／ 700（カード見出しh3）
  - Hero本文: 17px/2.05　／　セクションリード文: 15.5px/2.05　／　カード本文: 14–14.5px/1.95　／　ナビ: 14.5px　／　アイキャッチラベル: 13px, letter-spacing .14em

**Shape & Spacing**
- 角丸大きめ：カード24px、ヒーロー写真枠28px、ボタン・バッジは100px（ピル型）
- 装飾用の小さな円（ロゴのドットのモチーフ）をヒーロー画像の周囲に24–56px径で配置
- セクション余白：上下92–104px、左右56px。コンテンツ最大幅1120px中央寄せ
- カードグリッド：指導方針=2カラム(gap 28px)、特色講座=3カラム(gap 26px)

**Hover states（実装済みの参考値）**
- ナビリンク：`color` → `#B8860B`
- 主ボタン：`background` → `#242019`

## Pattern B — 静かな知性（Quiet Intelligence / Editorial）
**Concept**: 明朝の見出しと細い罫線、章立ての構成で、落ち着いた信頼感を伝える。装飾は最小限、余白と版組みで語る。

**Colors**
- 背景: `#F7F4EE`（ページ）／ `#F1EDE3`（特色講座セクションの帯）
- 本文テキスト: `#24262B`（メイン）／ `#5B5F66`（サブ）／ `#8B8D91`（注記）
- 罫線: `rgba(36,38,43,.14)`（細い区切り）／ `#24262B`（構造的な太罫線・グリッド上下の枠線）
- アイキャッチ・章番号のトーン: `#A67C00`（この案専用の落ち着いた金褐色。装飾ドットのみブランド原色 `#F8BC01 #F59084 #47BEE7 #7CC628` を使用）
- カード番号色（01/02/03）: `#B8860B`（金）／ `#D9604E`（珊瑚）／ `#1A93BE`（水色）
- 反転セクション（クレオスタディについて）: 背景 `#24262B`、文字 `#F7F4EE`、アクセント `#F8BC01`。Footerは反転せず地色のまま。

**Typography**
- 見出し（H1/H2）: **Shippori Mincho**, weight 600 のみ使用（明朝体）
  - Hero H1: 44px/1.75　／　セクションH2: 30px/1.7（About: 28px/1.8、Footer: 26px）
- アイキャッチ・章番号ラベル: **Noto Serif JP**, weight 400
- 本文: **Noto Sans JP**, weight 400（本文）／ 500・600・700（ボタン・強調テキスト）
  - 本文: 15–15.5px/2.1　／　ラベル: 12–13px, letter-spacing .12–.18em

**Shape & Spacing**
- 角丸なし（0px）。細い罫線と太い構造罫線のコントラストで構成
- 「CHAPTER I／II／III」「No. 01/02/03」のようなエディトリアルなラベリング
- ヒーロー写真：カードではなく、18px外側にオフセットした1pxアウトライン枠
- セクション余白：上下96–114px、左右60px。特色講座の3カラムはカードでなく縦罫線区切り

**Hover states**: 未実装（今回のリファレンスにはhoverスタイルを定義していません。実装時に標準的なフィードバック（下線の濃淡変化など）を追加することを推奨します）。

## Pattern C — 個性が、かたちになる（Bold Modern / Dot motif）
**Concept**: 大きな見出しと原色のドットモチーフ（ロゴのモチーフを拡張）で、自信を持って語る現代的な一枚。

**Colors**
- 背景: `#FFFFFF`（ページ）／ `#FBF8F3`（About セクションの帯）
- 反転（濃色）セクション: `#1C1B22` 背景・白文字 — 指導方針セクションとFooterの2箇所（A・Bより反転面が多く、コントラストが強い）
- 本文テキスト: `#1C1B22`（メイン）／ `#4A4950`, `#6B6A72`（サブ）
- ブランドドット：ヒーローバッジのconic-gradient（4色を90度ずつ）、ヒーロー画像角の4色ドットクラスタ、カードの上部8pxソリッドバー
- カードタグ色：金 `#8A6A00` on `#FEF3D6`／珊瑚 `#C0483A` on `#FCEAE7`／水色 `#0F6E8F` on `#EAF7FC`
- 指導方針パネル（濃色地）：金パネル `background: rgba(248,188,1,.12)` `border: rgba(248,188,1,.4)`／水色パネル `background: rgba(71,190,231,.12)` `border: rgba(71,190,231,.4)`

**Typography**
- 見出し（H1/H2）: **Zen Kaku Gothic New**, weight 900 のみ使用（極太ゴシック）
  - Hero H1: 58px/1.35, letter-spacing -.02em　／　セクションH2: 38px/1.4（About: 32px/1.55、Footer: 30px）
- 本文・ラベル: **Noto Sans JP**, weight 400（本文）／ 700（ラベル・カード見出し・ナビ）
  - 本文: 15.5–16.5px/1.9–2.05　／　トラッキングラベル: 13px, letter-spacing .1em（例：「POLICY ／ 指導方針」のような英日併記の小ラベル）

**Shape & Spacing**
- 角丸混在：ナビ・ボタン・バッジは100px（ピル型）、カード・ヒーロー画像は28–32px、カード上部は8px角なしソリッドバー
- セクション見出しは左寄せH2＋右寄せの小ラベルで非対称に配置（`justify-content:space-between`）
- セクション余白：上下92–104px、左右56px。ヒーローの見出し・本文列は最大幅760pxに制限

**Hover states**: 未実装（Bと同様、今回のリファレンスにはhoverスタイルを定義していません）。

## Interactions & Behavior
- ヘッダーのナビリンク・各CTAは同一ページ内のアンカーリンク（`href="#pt-policy"` など）でスクロールします。ページ遷移やJSルーティングはありません。
- フッターの「お問い合わせ」「無料体験・資料請求」ボタンは現状 `href="#"` のプレースホルダーです。実装時に実際の問い合わせ導線（フォーム／電話／メール等）に接続してください。
- 「姉妹校：上野入谷教室」リンクは実在する外部ページ `https://www.creo-study.com/ueno` を指しています。新規ページの実装は不要です。
- ヒーローのビジュアル枠は現在プレースホルダー（カメラアイコン＋「三田教室の写真を」の文字）です。実写真が用意でき次第、同じ枠サイズ・角丸で差し替えてください。
- ホバーステートはA案のみ定義済み（上記参照）。B・C案は未定義のため、実装時に一貫したインタラクションフィードバックを追加することを推奨します。

## State Management
