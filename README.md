## Overview
学習塾「クレオスタディ」三田教室のトップページを刷新するデザイン提案です。既存サイト（creo-study.com）のロゴ・世界観は維持しつつ、情報量を拡充した動的なワンページ構成に書き換えています。

- 指導方針：少人数グループ指導 + 個別指導で、一人ひとりに合った最適な学びを提供
- コース紹介：実際の三田教室ページ（creo-study.com/mita）に掲載されている5コース（アドバンスドプログラム／パズル算数／読みとく国語／個別指導1対1／個別指導1対2）

姉妹校である上野入谷教室は、フッターからの外部リンク1本のみで案内しています（`https://www.creo-study.com/ueno` — 既存の実ページ）。Topics・お問い合わせ系ボタンは既存サイトの実ページ（`/contact/`, `/contact-trial/`, `/contact-request/`, `/blog` ほか）への外部リンクとして接続済みです。

**3案は同じ1ページ（同じ構成・同じコピー）に対する、3つの異なる視覚的方向性です。** クライアントが方向性を選定するための比較材料であり、3つとも本番実装する想定ではありません。どれか1案が選ばれたら、その1案だけを実装してください。

## About the Design Files
このバンドルに含まれるHTMLファイルは **デザインリファレンス（参考実装）** です。本番のコードとしてそのまま流用するものではありません。既存のプロダクトコードベース（React / Vue / WordPress など）がある場合は、そのコードベースが既に採用しているパターン・コンポーネント・ライブラリを使って、このデザインを再構築してください。まだ環境がない場合は、プロジェクトに最も適したフレームワークを選定した上で実装してください。

各HTMLは **クラスベースのCSS（`<style>`ブロック）+ 素のJavaScript（`<script>`ブロック）** で書かれた自己完結ファイルです。デザイントークン（色・モーション値）は各ファイル冒頭の `:root` のCSSカスタムプロパティにまとまっています。

## Interactions & Behavior（3案共通・実装済み）
- **スクロール連動リビール**: `.rv` クラスの要素が IntersectionObserver（threshold 0.12 / rootMargin 下端 -7%）で画面に入るとフェード+上昇表示。`[data-stagger]` を付けたグループは子要素が順にディレイ表示。一度表示したら再スクロールでは再生しない（`ONCE = true`）。
- **スクロール追従ヘッダー**: `scrollY > 24` で `.scrolled` クラスが付き、パディングとロゴが縮小、影が付く。
- **ページ内スムーズスクロール**: CSS `scroll-behavior: smooth` + 各セクションの `scroll-margin-top` で固定ヘッダー分を確保。
- **モバイルナビ**: 960px以下でハンバーガー。開閉は `body.nav-open`。リンククリックで自動的に閉じる。
- **トップへ戻るボタン**: `scrollY > 600` で表示。
- **ホバーステート**: ナビ・ボタン・カード・Topics行・地図に定義済み（各案のCSS参照）。
- **アクセシビリティ**: `prefers-reduced-motion: reduce` 環境では全アニメーション・スムーズスクロールを無効化し、コンテンツを即時表示。
- **C案のみ**: 画面上部にブランド4色グラデーションのスクロール進捗バー。

## Motion（案ごとの性格付け）
移動量・時間・イージングは各ファイルの `:root`（`--rv-y` / `--rv-t` / `--rv-e`）、時差は `<script>` 内の `STAGGER` で定義。

| | A やわらかな余白 | B 静かな知性 | C 個性が、かたちになる |
|---|---|---|---|
| 上昇距離 `--rv-y` | 26px | 18px | 36px |
| 時間 `--rv-t` | .75s | .9s | .6s |
| イージング `--rv-e` | ease-out系 (.25,.46,.45,.94) | 長い減速 (.19,1,.22,1) | スナップ系 (.16,1,.3,1) |
| 時差 `STAGGER` | 90ms | 110ms | 70ms |
| 固有の動き | ヒーローの装飾ドットがふわふわ浮遊 | 静的（動きは最小限） | バッジの4色ホイール回転・ドットのパルス・進捗バー |

## Pattern A — やわらかな余白（Soft Light / Airy Warmth）
**Concept**: 丸みと余白で、やさしく迎える。ロゴの4色（金・珊瑚・水色・緑）をそのまま柔らかく使う、あたたかい印象。

- 背景 `#FBF8F3`／カード `#FFFFFF`／本文 `#3A3530`・`#655D53`・`#6B6154`／罫線 `#EEE6D8`
- ブランドドット原色: 金 `#F8BC01`／珊瑚 `#F59084`／水色 `#47BEE7`／緑 `#7CC628`。テキスト用濃色: `#B8860B` `#D9604E` `#1A93BE` `#5C9418`
- パネル淡色: 金系 `#FEF3D6`／水色系 `#EAF7FC`／珊瑚系 `#FCEAE7`。反転セクション（About）: 背景 `#3A3530`
- 見出し: **M PLUS Rounded 1c / 800**（H1 50px、H2 34px）。本文: Noto Sans JP
- 角丸大きめ（カード24px、写真28px、ボタン100pxピル）。セクション見出しは中央寄せ
- カードhover: 6px浮上 + 大きめの柔らかい影。ヒーロー装飾ドットは浮遊アニメーション

## Pattern B — 静かな知性（Quiet Intelligence / Editorial）
**Concept**: 明朝の見出しと細い罫線、章立ての構成で、落ち着いた信頼感を伝える。装飾は最小限、余白と版組みで語る。

- 背景 `#F7F4EE`／帯 `#F1EDE3`／本文 `#24262B`・`#5B5F66`・`#8B8D91`／罫線 `rgba(36,38,43,.14)` + 構造罫線 `#24262B`
- アイキャッチ・章番号: `#A67C00`（金褐色）。カード番号: `#B8860B` `#D9604E` `#1A93BE`。反転セクション（About）: 背景 `#24262B`
- 見出し: **Shippori Mincho / 600**（H1 44px、H2 30px）。ラベル: Noto Serif JP。本文: Noto Sans JP

## Pattern C — 個性が、かたちになる（Bold Modern / Dot motif）
**Concept**: 大きな見出しと原色のドットモチーフ（ロゴのモチーフを拡張）で、自信を持って語る現代的な一枚。

- カードタグ: 金 `#8A6A00` on `#FEF3D6`／珊瑚 `#C0483A` on `#FCEAE7`／水色 `#0F6E8F` on `#EAF7FC`
- 見出し: **Zen Kaku Gothic New / 900**（H1 58px、H2 38px、letter-spacing -.02em）。本文: Noto Sans JP
- 角丸混在（ピル100px、カード28px、カード上部8pxソリッドバー）。セクション見出しは左H2 + 右英日ラベルの非対称配置
